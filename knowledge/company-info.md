# AIMED Project

AIMED stands for AI-Managed Enterprise Dashboard.

## Team
- Project Lead: Jisu Park
- AI: Claude Code (Anthropic)

## Goals
1. Build a local-first AI secretary
2. Support multi-platform messaging (Discord, Telegram, Slack)
3. Provide task management, reminders, and reports
4. RAG-powered knowledge base for company docs
5. Voice input support

## Tech Stack
- TypeScript + Node.js 22
- SQLite for storage
- Claude Code CLI as LLM provider
- Inspired by OpenClaw architecture
